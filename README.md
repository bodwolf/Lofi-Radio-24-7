# 📻 Lofi Radio 24/7

A premium web-based Lofi music player featuring a **Glassmorphism UI**, real-time live stream integration, and interactive animations. Built for focus, studying, and relaxation.

## 🚀 Live Demo
**[Click here to listen now!](https://bodwolf.github.io/Lofi-Radio-24-7/)**

## ✨ Features
* **📺 24/7 Live Streams:** Integrated with YouTube API to fetch continuous Lofi broadcasts.
* **🔄 Multi-Channel Support:** Switch between Study, Gaming (Synthwave), and Sleep modes.
* **💿 Interactive UI:** Rotating glass disc animation that syncs with playback state.
* **🎨 Aesthetic Design:** Modern Glassmorphism effect with responsive layouts.
* **🎛️ Volume Control:** Custom slider for audio adjustment.
* **🌧️ Ambience Effects:** Toggle between Noise and Rain effects for better immersion.

## 🛠️ Technologies Used
* **HTML5 & CSS3:** For structure and styling (Pure CSS animations).
* **JavaScript (ES6):** For logic and state management.
* **YouTube IFrame API:** For streaming audio content.
